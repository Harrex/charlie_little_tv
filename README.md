# Charlie and the Chocolate Factory - Little TV Hands

## Install
Clone this repo to the desktop (this should already be done)
Add video files (with exact capitalisation etc):
- `Audrey.mp4`
- `Harinee.mp4`
- `Heidi.mp4`
- `static.mp4`

## Usage
To run, either double-click on `Audrey.command` or `Harinee.command` on the Desktop (depending on actor). A fullscreen window should open.
To continue to the next clip, press any letter key on the keyboard. The TV will return to static after each clip finishes.

The 2 videos will loop forever, so if a key is accidentally pressed, press again to get back.

## Editing the script
The script can be edited remotely and synced if the computer has internet connection. Videos must be transferred manually.
